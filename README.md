# 西安交通大学软件学院编译原理实验

## 简介
本实验一共有三部分：

* lab-1: xjtu\_lexer, 词法分析
* lab-2: xjtu\_parser, 语法分析
* lab-3: xjtu\_codegen, 中间代码生成

本实验使用了自己实现的语法分析器，没有借助分析器生成工具。
